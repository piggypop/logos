```python
primes = [number for number in range(2, 101) if all(number % i != 0 for i in range(2, int(number**0.5) + 1))]

```

*   **`number for number in ...`:**  Αυτό είναι το βασικό τμήμα της list comprehension.  Για κάθε `number` στην ακολουθία που δημιουργήθηκε από το `range()`, θα εκτελεστεί ο επόμενος έλεγχος.
*   **`if all(number % i != 0 for i in range(2, int(number**0.5) + 1))`:** Αυτός είναι ο έλεγχος για να δούμε αν ο αριθμός είναι πρώτος.
    *   **`range(2, int(number**0.5) + 1)`:**  Ελέγχει μόνο τους αριθμούς μέχρι την τετραγωνική ρίζα του `number`.  Αν ένας αριθμός έχει έναν διαιρέτη μεγαλύτερο από την τετραγωνική του ρίζα, τότε έχει σίγουρα και έναν μικρότερο.
    *   **`number % i != 0`:** Ελέγχει αν ο `number` διαιρείται με το `i`.  Αν το υπόλοιπο δεν είναι 0, τότε ο `i` δεν είναι διαιρέτης του `number`.
    *   **`all(...)`:**  Επιστρέφει `True` μόνο αν όλες οι συνθήκες στο `number % i != 0` είναι `True`.  Δηλαδή, αν ο `number` δεν διαιρείται με κανέναν αριθμό από το 2 μέχρι την τετραγωνική του ρίζα.
