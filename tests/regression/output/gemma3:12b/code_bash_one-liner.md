Εντάξει, εδώ είναι μια εντολή bash που μετράει τις γραμμές κώδικα Python σε όλα τα `.py` αρχεία ενός φακέλου αναδρομικά:

```bash
find . -name "*.py" -print0 | xargs -0 wc -l
```

**Επεξήγηση:**

*   `find . -name "*.py"`:  Βρίσκει όλα τα αρχεία με επέκταση `.py` ξεκινώντας από τον τρέχοντα φάκελο (`.`) και αναδρομικά.
*   `-print0`:  Εκτυπώνει τα ονόματα των αρχείων χωρισμένα με null χαρακτήρες. Αυτό είναι σημαντικό για να χειριστεί σωστά ονόματα αρχείων που περιέχουν κενά ή ειδικούς χαρακτήρες.
*   `xargs -0 wc -l`:  Περνάει τα ονόματα των αρχείων (που δημιουργήθηκαν από το `find`) στο `wc -l`.  Το `wc -l` μετράει τις γραμμές κάθε αρχείου. Το `-0` στο `xargs` διασφαλίζει ότι τα ονόματα αρχείων με null χαρακτήρες χειρίζονται σωστά.
